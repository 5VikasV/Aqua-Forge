# Aqua Forge User Flow

## Goal

Help a developer go from:

"I have an idea."

to

"I know exactly what to build."

within a few minutes.

## User Journey

1. Enter a project idea.
2. Refine the idea with AI research.
3. Generate a complete project blueprint.
4. Start building with confidence.

## Idea Forge Flow

1. User enters an idea.
2. Aqua Forge asks questions to understand the idea.
3. Aqua Forge researches existing solutions.
4. Aqua Forge identifies opportunities and improvements.
5. Aqua Forge confirms the refined idea.
6. Aqua Forge generates the engineering blueprint.
7. User reviews and exports the blueprint.
