# Aqua Forge UI

## Screen 1 - Home

Purpose:

Help users immediately start transforming an idea into a structured software project.

Primary Action:

Enter a project idea and click "Start Forging".

### Homepage Sections

- Hero Section
- Project Idea Input
- Start Forging Button
- Trending Ideas
- Personalized Recommendations
- Hackathon Ideas

## Core Workflow

Aqua Forge follows a guided engineering workflow instead of a traditional AI chat.

### Stage 1
💡 Idea

The user describes their project idea.

↓

### Stage 2
🔍 Research

Aqua Forge researches existing solutions, competitors, and identifies opportunities.

↓

### Stage 3
🤔 Refinement

Aqua Forge asks targeted questions to better understand the project.

↓

### Stage 4
📋 Planning

Generate features, requirements, user stories, and milestones.

↓

### Stage 5
🏗 Architecture

Generate the system architecture, database design, APIs, and folder structure.

↓

### Stage 6
🚀 Blueprint

Generate the final engineering blueprint ready for development.
